Renegade BBS

Origin

Renegade is an open source (ↄ) bulletin board system (BBS) written for IBM PC-compatible computers running MS-DOS that gained popularity among hobbyist BBSes in the early to mid 1990s. It was originally written by Cott Lang in Pascal, based on the source code of Telegard, which was in turn based on the earlier WWIV.
